Aggiunta di un docuemnto:

Le regole sono le seguenti:

1. Ogni file PDF deve avere un nome univoco. (esempio: integrali_1.pdf)

2. Ogni file PDF soluzione associata, deve avere un nome univoco e contenere nel nome del PDF la parola "soluzione" (esempio: soluzione_integrali_1.pdf)

3. Ogni titolo che dai alla sezione alla creazione deve essere univoco. (esempio: Integrali di Analisi 1)
