Aggiungi o rimuovi documenti

Le regole sono le seguenti:

1. Ogni PDF deve avere un nome univoco. (esempio: prova_1.pdf)

2. Ogni soluzione associata deve avere un nome univoco e contenere nel nome del PDF la parola "soluzione" (esempio: soluzione_prova_1.pdf)

3. Per eliminare il file inserire il nome del file completo (esempio: prova_1.pdf), la soluzione e eliminata automaticamente.
